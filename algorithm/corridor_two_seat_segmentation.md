# Corridor Two-Seat Segmentation

目標：給定字串 `corridor`（只含 `S` 座位與 `P` 植物），計算將走廊切成不重疊區段且每段**恰有兩個座位**的切法數，答案取模 `MOD = 1_000_000_007`。

## 核心想法

- 只要知道每兩個座位間的植物數量，即可決定分隔線的擺放方式。
- 設所有座位位置為 `pos[0..m-1]`（`m` 為座位總數，需為偶數且至少 2，否則答案為 0）。區段應為 `[pos[0], pos[1]]`, `[pos[2], pos[3]]`, …。
- 在 `pos[2k+1]` 與 `pos[2k+2]` 之間共有 `gap_k = pos[2k+2] - pos[2k+1] - 1` 個植物，分隔線可以放在這個區間的 `gap_k + 1` 個位置之一，互相獨立，故總切法為 `∏ (gap_k + 1)`（`k` 走遍所有相鄰區段邊界）。

## 步驟

1. 掃描字串收集所有座位索引 `pos`。若座位數 `m` 為 0 或奇數，回傳 0。
2. 對每個區段邊界 `k = 0..(m/2 - 2)`，計算 `gap_k = pos[2k+2] - pos[2k+1] - 1`，累乘 `ans = ans * (gap_k + 1) % MOD`。
3. 其餘情況：若只有一組座位 (`m == 2`)，沒有可放分隔線的位置，答案為 1。

## 複雜度

- 時間：O(n)，單次掃描並計算間距。
- 空間：O(m) 儲存座位位置。

## 邊界與防呆

- 無座位 / 座位數為奇數 → 0。
- 只有兩個座位 → 1。
- 連續座位（間距 0）→ `gap + 1 = 1`，不影響乘積。
- 需使用 64 位整數進行乘法後再取模，避免中途溢位。

## 小型測試

- `"SSPPSPS"`：座位索引 `[0,1,4,6]`，間距 `(4-1-1)=2` → `2+1=3`，答案 3。
- `"PPSPSP"`：座位 `[2,4]`，無間隔，答案 1。
- `"S"`：座位數為奇數，答案 0。
