# Svelte 5 Series README

## Summary

- Audience: beginner to practical level.
- Stack baseline: TypeScript + SvelteKit 2.51.x + Svelte 5.50.x.
- Baseline date: 2026-02-14.
- Learning mode: each batch covers 3 chapters with hands-on tasks.

## Batch Learning Route (8 batches)

- Batch 1: `05`, `06`, `07` (events/effects/state management)
- Batch 2: `08`, `09`, `10` (composition/styling/SvelteKit bootstrap)
- Batch 3: `11`, `12`, `13` (data loading/form actions/SSR streaming)
- Batch 4: `14`, `15`, `16` (advanced routing/errors/performance)
- Batch 5: `17`, `18`, `19` (testing/deploy/upgrade governance)
- Batch 6: `20`, `21`, `22` (migration/ecosystem/authentication)
- Batch 7: `23`, `24`, `25` (accessibility/i18n/advanced TypeScript)
- Batch 8: `26` (capstone product delivery)

## Estimated Time Per Chapter

- Must-read chapters: 90-120 min each.
- Skippable chapters: 75-90 min each.
- Recommended cadence: 3 chapters per week + 1 review day.

## Reading Priority

- Must-read (`必讀`): `00`, `01`, `02`, `03`, `04`, `05`, `06`, `10`, `11`, `12`, `13`, `17`, `19`, `22`
- Recommended (`建議`): `20`, `21`, `23`, `25`
- Optional (`可跳讀`): `07`, `08`, `09`, `14`, `15`, `16`, `18`, `24`
