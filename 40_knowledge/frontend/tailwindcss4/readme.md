# Tailwind CSS 4 Series README

## Summary

- Audience: beginner to practical level.
- Stack baseline: Tailwind CSS 4.1.x (基礎 HTML + Vite 6.x, React 整合: React 19.2.x + Next.js 16, Svelte 整合: Svelte 5.50.x + SvelteKit 2.51.x).
- Baseline date: 2026-02-14.
- Learning mode: each batch covers 3 chapters with hands-on tasks.

## Batch Learning Route (8 batches)

- Batch 1: `01`, `02`, `03` (哲學、安裝、色彩)
- Batch 2: `04`, `05`, `06` (字型、間距、邊框)
- Batch 3: `07`, `08`, `09` (Flexbox、Grid、定位)
- Batch 4: `10`, `11`, `12` (響應式、狀態變體、深色模式)
- Batch 5: `13`, `14`, `15` (動畫、漸層/濾鏡、容器查詢)
- Batch 6: `16`, `17`, `18` (@theme、@utility、插件)
- Batch 7: `19`, `20`, `21` (效能、React 整合、Svelte 整合)
- Batch 8: `22`, `23`, `24` (設計系統、v3 遷移、升級治理)

## Estimated Time Per Chapter

- Must-read chapters: 90-120 min each.
- Suggested chapters: 60-90 min each.
- Skippable chapters: 45-75 min each.
- Recommended cadence: 3 chapters per week + 1 review day.

## Reading Priority

- Must-read (`必讀`): `00`, `01`, `02`, `03`, `05`, `07`, `10`, `11`, `12`, `16`, `19`, `20`, `24`
- Suggested (`建議`): `04`, `06`, `08`, `13`, `17`, `21`, `22`, `23`
- Skippable (`可跳讀`): `09`, `14`, `15`, `18`
