# q230. Kth Smallest Element in a BST / BST 第 K 小元素
[230. Kth Smallest Element in a BST](https://leetcode.com/problems/kth-smallest-element-in-a-bst/)

Medium

## 解題筆記與程式碼

通用演算法筆記：[bst_inorder_sorted_sequence](../algorithm/bst_inorder_sorted_sequence.md)
資料結構筆記：[binary_search_tree](../data_structure/binary_search_tree.md)

### 思路拆解

- BST 中序遍歷是遞增序列。
- 走到第 `k` 個訪問到的節點即可返回。

### Rust 參考

```rust
impl Solution {
    pub fn kth_smallest(root: Option<Rc<RefCell<TreeNode>>>, k: i32) -> i32 {
        fn dfs(node: &Option<Rc<RefCell<TreeNode>>>, k: &mut i32, ans: &mut i32) {
            if node.is_none() || *k == 0 { return; }
            let rc = node.as_ref().unwrap();
            let (left, val, right) = {
                let n = rc.borrow();
                (n.left.clone(), n.val, n.right.clone())
            };
            dfs(&left, k, ans);
            if *k > 0 {
                *k -= 1;
                if *k == 0 {
                    *ans = val;
                    return;
                }
            }
            dfs(&right, k, ans);
        }

        let mut kk = k;
        let mut ans = 0;
        dfs(&root, &mut kk, &mut ans);
        ans
    }
}
```
