# q190. Reverse Bits / 顛倒位元
[190. Reverse Bits](https://leetcode.com/problems/reverse-bits/)

Easy

Topics

Companies

Given a 32-bit unsigned integer `n`, reverse its bits and return the result.
給定一個 32-bit 無號整數 `n`，請將其位元順序反轉並回傳結果。

**Example 1:**

**Input:** n = `00000010100101000001111010011100`  
**Output:** `00111001011110000010100101000000`
**Explanation:** The input bit string represents unsigned integer 43261596, so return 964176192.
**說明：**輸入位元字串代表無號整數 43261596，因此回傳 964176192。

**Example 2:**

**Input:** n = `11111111111111111111111111111101`  
**Output:** `10111111111111111111111111111111`
**Explanation:** The input bit string represents unsigned integer 4294967293, so return 3221225471.
**說明：**輸入位元字串代表無號整數 4294967293，因此回傳 3221225471。

**Constraints:**

- The input is a binary string of length 32.
- 輸入是一個長度為 32 的二進位字串。

**Follow up:** If this function is called many times, how would you optimize it?
**進階：**若此函式會被重複呼叫很多次，要如何優化？

## 解題筆記與程式碼

通用演算法筆記：[bit_reversal_shift_mask](../algorithm/bit_reversal_shift_mask.md)
資料結構筆記：[byte_reverse_lookup_table](../data_structure/byte_reverse_lookup_table.md)

### 思路拆解

- 從 `n` 最低位開始，每次取出最低位（`n & 1`），依序補到答案 `ans` 的最低位。
- 做法是先把 `ans` 左移一位，再把剛取出的 bit OR 進去：`ans = (ans << 1) | (n & 1)`。
- 然後把 `n` 右移一位，重複 32 次即可完成整體反轉。

### 演算法

1. `ans = 0`。
2. 重複 32 次：
   - `ans <<= 1`
   - `ans |= n & 1`
   - `n >>= 1`
3. 回傳 `ans`。

### 時間與空間

- 時間：`O(32)`，固定長度可視為 `O(1)`。
- 空間：`O(1)`。

```rust
impl Solution {
    pub fn reverse_bits(mut x: u32) -> u32 {
        let mut ans = 0u32;
        for _ in 0..32 {
            ans = (ans << 1) | (x & 1);
            x >>= 1;
        }
        ans
    }
}
```
