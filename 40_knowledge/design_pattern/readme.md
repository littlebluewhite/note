# Design Pattern Series README

## Summary

- Audience: developers with Go/Rust fundamentals, targeting intermediate design skill.
- Language baseline: Go 1.24+ / Rust 2024 edition.
- Baseline date: 2026-02-17.
- Learning mode: each batch covers 2-3 patterns with dual-language implementation.

## Batch Learning Route (11 batches)

- Batch 1: `01`, `02`, `03` (Creational 基礎 — Singleton, Factory Method, Abstract Factory)
- Batch 2: `04`, `05` (Creational 進階 — Builder, Prototype)
- Batch 3: `06`, `07`, `08` (Structural 基礎 — Adapter, Bridge, Composite)
- Batch 4: `09`, `10`, `11` (Structural 中階 — Decorator, Facade, Flyweight)
- Batch 5: `12`, `13`, `14` (Structural→Behavioral 過渡 — Proxy, Chain of Resp, Command)
- Batch 6: `15`, `16`, `17` (Behavioral 中階 — Iterator, Mediator, Memento)
- Batch 7: `18`, `19`, `20` (Behavioral 核心 — Observer, State, Strategy)
- Batch 8: `21`, `22`, `23` (Behavioral 進階 — Template Method, Visitor, Interpreter)
- Batch 9: `24`, `25`, `26` (Modern Go/Rust — Functional Options, Newtype, Typestate)
- Batch 10: `27`, `28` (Modern 架構 — Repository, Middleware)
- Batch 11: `29`, `30`, `31` (Modern 並行/容錯 — Worker Pool, Circuit Breaker, RAII)

## Estimated Time Per Pattern

- Must-read patterns: 60-90 min each.
- Skippable patterns: 45-60 min each.
- Recommended cadence: 2-3 patterns per week + 1 review day.

## Reading Priority

- Must-read (`必讀`): `01`, `02`, `04`, `06`, `08`, `09`, `10`, `18`, `19`, `20`, `21`, `24`, `25`, `29`
- Skippable (`可跳讀`): `03`, `05`, `07`, `11`, `12`, `13`, `14`, `15`, `16`, `17`, `22`, `23`, `26`, `27`, `28`, `30`, `31`
